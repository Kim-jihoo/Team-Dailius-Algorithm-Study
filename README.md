## 스터디 규칙
- 주말은 휴식!
- 모르면 그냥 물어보기!
- 날짜에 맞는 문제를 풀고, 해결 과정을 기록하여 `markdown`이나 `링크`로 코드와 함께 `풀 리퀘스트`한다.
### 풀 리퀘스트
- 한 주 분량을 한 주에 한 번 `풀 리퀘스트`한다.
- `merge`는 해당 주 `토요일`에 진행한다.
### 벌금
- 문제를 해결하지 못한 합당한 이유가 없을 경우, 벌금(1500원)을 낸다.\
